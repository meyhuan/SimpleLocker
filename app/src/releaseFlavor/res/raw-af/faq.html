Hoe kan ek hierdie app verwyder?
Gaan na Instellings > Sekuriteit > Toestelbestuurders en disable AcDisplay. You can now verwyder AcDisplay.

Hoe kan ek die permanente kennisgewings van die aktiewe modus van die vergrendelingsscherm deaktiveer?
The notification is necessary to ensure AcDisplay isn't killed by the system. However, to hide it safely (sending test notification won't function anymore!) open app info en skakel: "kennisgewings vertoon" uit.

When will the secure lockscreen (pattern/PIN) be implemented?
It is not possible to create a custom, secure lockscreen for Android. Other apps that have this feature only pretend to be secure, but in the real world there's always a way to bypass them.

When will the double-tap to wake be implemented?
It is not possible to implement, cause it requires a kernel modification.
